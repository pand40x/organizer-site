# organizer-site

Public companion site for the Organizer macOS app. The application
source repository is private; this repository exists solely to host
the public privacy policy and support page the Mac App Store
requires. The site is published from this repo's root via GitHub
Pages and lives at:

<https://pand40x.github.io/organizer-site/>

| Page | URL |
|---|---|
| Landing | <https://pand40x.github.io/organizer-site/> |
| Privacy policy | <https://pand40x.github.io/organizer-site/privacy/> |
| Support | <https://pand40x.github.io/organizer-site/support/> |

The pages are plain Jekyll markdown rendered with the `minima`
theme. Editing one of the markdown files in this repo and pushing
to `main` triggers GitHub Pages to rebuild within a minute.

Source code, release notes, internal docs, and the issue tracker
all live in the private repo.
